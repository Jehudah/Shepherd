## üÜò Common Issues & Solutions

### Firebase Connection Failed
**Problem**: App can't connect to Firebase
**Solution**:
- Verify GoogleService-Info.plist is in project root
- Check google-services.json is in android/app/
- Rebuild app with `eas build`
- See FIREBASE-SETUP-COMPLETE.md Part 8

### Build Failed on EAS
**Problem**: Build fails during Expo build process
**Solution**:
- Clear cache: `eas build --clear-cache`
- Check package.json for conflicts
- Verify all dependencies installed
- See EXPO-DEPLOYMENT-GUIDE.md Troubleshooting

### App Store Rejection
**Problem**: Apple rejected your app
**Solution**:
- Read rejection email carefully
- Common issues: demo account, screenshots, privacy
- Fix issues mentioned
- Resubmit through App Store Connect
- See IOS-APPSTORE-GUIDE.md Part 9

### Play Store Publishing Delayed
**Problem**: App stuck in "Publishing" status
**Solution**:
- Wait 2-24 hours (normal delay)
- Check for emails from Google
- Verify all dashboard tasks complete
- See ANDROID-PLAYSTORE-GUIDE.md Part 8
